# kmake

## Requrements
* node.js >=12

## Usage
* `npm i -g kmake`
* `kmake your_kmake.yml your_project_template output_dir`

## Dev
* `npm run dev-mac`

## Plaftorm settings
you can define platform settings for:
* `defines`
* `includePaths`
* `dependencies`
* `hooks`
* `buildFlags`
* `linkerFlags`
